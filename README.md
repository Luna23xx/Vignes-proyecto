# Sitio Web Vignes – Proyecto de Desarrollo Web

Este proyecto consiste en el desarrollo de un sitio web informativo enfocado en la presentación de una marca relacionada con el mundo vitivinícola. El objetivo principal es ofrecer una experiencia visual atractiva y una navegación clara para los usuarios.

## Tecnologías utilizadas
- HTML
- CSS

## Objetivo del proyecto
Desarrollar un sitio web que permita presentar información de manera estructurada y atractiva, mejorando la presencia digital de una marca.

## Funcionalidades
- Diseño responsivo adaptable a distintos dispositivos
- Navegación clara e intuitiva
- Estructuración de contenido mediante secciones
- Diseño visual enfocado en la experiencia del usuario
